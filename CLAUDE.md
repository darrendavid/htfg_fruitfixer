# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is the **HawaiiFruit.net** static website archive for the Hawaii Tropical Fruit Growers (HTFG). It is a content-heavy, image-rich educational site about tropical fruit cultivation on Hawaii's Big Island, spanning nearly 30 years of documentation.

This is **not** a software development project — there is no build system, test suite, or CI/CD pipeline. It is a collection of static HTML, images, documents, and photo galleries being transformed into a structured plant database.

## Key Technical Details

- **HTML version**: HTML 4.01 Transitional (legacy, not HTML5)
- **Gallery structure**: Most photo galleries use Adobe Web Photo Gallery format with iframe-based layouts (`index.html`, `ThumbnailFrame.html`, `/images/`, `/pages/`, `/thumbnails/`)
- **JavaScript**: Minimal — 31 JS files, mostly auto-generated from PowerPoint/Office exports
- **CSS**: 103 files, largely per-gallery stylesheets generated by Photoshop
- **No backend**: Purely static content, no server-side processing
- **Directory names contain spaces**: Many paths have spaces (e.g., `HawaiiFruit. Net`, `fruit pix`, `2007 fruit shoot`) — always quote paths in shell commands
- **Node.js version**: v24.14.0 (uses ES modules via .mjs extension since package.json has `"type": "commonjs"`)

## Content Sections

Major sections linked from the homepage include: avocado research, 12 Trees Project (USDA-funded), stone fruit trials, fig variety tests, culinary school recipes, fruit data sheets, Japan fruit information, HTFG conference materials, publications, gluten-free resources, and educational workshop PowerPoints.

## Known Issues & Workarounds

- **Bun runtime crash**: The Claude Code agent runtime (Bun v1.3.10 on Windows) crashes with "switch on corrupt value" panic when processing large directories (41GB+). Workaround: split work into smaller parallel batches (~27 dirs each) or use Node.js scripts directly via Bash instead of subagents.
- **Agent Bash access**: Subagents sometimes can't execute Bash commands. Solution: agents write scripts, then the main session runs them.
- **xlsx ESM import**: `import * as XLSX from "xlsx"` doesn't work in .mjs files. Fix: use `createRequire(import.meta.url)` to require xlsx.
- **pdf-parse version**: Must use v1.1.1 (not v2.x+). v2 has a completely different class-based API (PDFParse class with load/getText). v1 exports a simple `pdf(buffer)` function.
- **4 files with problematic whitespace**: `10  papaya.jpg` (double space), `sharwillgsm3 .jpg` (trailing space) — these cause `du` errors but don't affect processing.

## Working with This Repository

- **All internal links use absolute URLs** (`http://www.hawaiifruit.net/...`) — any link refactoring would need to convert these to relative paths
- **Image-heavy**: 22,981 images (19,400 JPG, 3,034 GIF, 545 PNG, 350 PSD)
- **`UserSelections.txt`** files in gallery directories contain Photoshop gallery generation metadata (numeric parameters), not user-facing content
- **`desktop.ini`** files are Windows Explorer folder customization metadata and can be ignored
- **Reference data**: `docs/reference/tropical_fruits_v10.csv` — external tropical fruit reference list
